# Install all these on Windows
winget install -e --id Kitware.CMake

# Build instructions with build.bat
- Initial setup: 
./build.bat configure
- Compile code: 
./build.bat build
- Run compiled exe: 
./build.bat run

- Remove the build folder: 
./build.bat clean
- Clean and recompile all code:
./build.bat rebuild
